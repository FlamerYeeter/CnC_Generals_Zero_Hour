◆ NameKeyType

enum NameKeyType

Note that NameKeyType isn't a "real" enum, but an enum type used to enforce the fact that NameKeys are really magic cookies, and aren't really interchangeable with ints. NAMEKEY_INVALID is always a legal value, but all other values are dynamically determined at runtime. (The generated code is basically identical, of course.)
