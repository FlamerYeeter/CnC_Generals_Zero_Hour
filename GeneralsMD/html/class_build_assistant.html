enum BuildAssistant::LocalLegalToBuildOptions
Enumerator
TERRAIN_RESTRICTIONS 

Check for basic terrain restrictions.

CLEAR_PATH 

Must be able to path find to location.

NO_OBJECT_OVERLAP 

Can't overlap enemy objects, or locally controled objects that can't move out of the way.

USE_QUICK_PATHFIND 

Use the quick pathfind method for CLEAR_PATH.

SHROUD_REVEALED 

Check to make sure the shroud is revealed.

NO_ENEMY_OBJECT_OVERLAP 

Can't overlap enemy objects only.

IGNORE_STEALTHED 

Units that we can't see are legal to "build" on. (when moving mouse around)

FAIL_STEALTHED_WITHOUT_FEEDBACK 

USE WITH IGNORE_STEALTHED except it will fail without BIB feedback (when clicking to place).

◆ buildTiledLocations()

BuildAssistant::TileBuildInfo * BuildAssistant::buildTiledLocations ( const ThingTemplate * thingBeingTiled,
Real angle,
const Coord3D * start,
const Coord3D * end,
Real tilingSize,
Int maxTiles,
Object * builderObject )
virtual

tiling wall object helper function, we can use this to "tile" walls when building

Tiling wall object helper function, we can use this to "tile" walls when building. This function will fill out the ARRAY world positions named 'positions' starting at the 'start' point in the world and going to the 'end' point in the world. No more than 'maxTiles' will be considered and if the position of any one of the objects in the chain is illegal the chain stops there. The number of objects actually used for the tiling is returned along with a pointer to the array of positions in the tile build info.

REQUIRES: Note that the array at 'postions' must be large enough to hold 'maxTiles' entries of positions
